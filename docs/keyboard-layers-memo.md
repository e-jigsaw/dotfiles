## Layer 1（ベース・80キー）

QWERTY配列ベース。気になる箇所だけ抜粋:
- [32] Aの左 = LCtrl（CapsLockをCtrlに）
- [45] L の右 = =（通常 ;）
- [46] 通常 ' の位置 = -
- [47] Enter手前 = ;
- [66] [67] 左親指 = LAlt / LGui
- [68] Space
- [70] BkSp
- [71] 不明コード #51770（おそらくレイヤーシフト系の親指キー、要確認）
- [72] Enter
- [74] [75] = LockL2 / LockL3（親指でレイヤー切替）
