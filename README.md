# Usage

## tmux keybind

| key | description |
| --- | ----------- |
| C-z | Meta        |
| r   | reload config |
| n   | new window  |
| d   | vertical split |
| D   | horizonal split |
| hjkl| move pane (like vim) |
| m   | choose window |
| w   | close window |
| W   | close pane  |
